# Learning-Go

This repository contains several small projects which are my first steps in the `GO` language

### 00-bookstore

A simple CLI application where you can add, update, view or delete entries that are saved locally as `JSON`

### 01-bookstore-api

A much more complicated version of `00-bookstore`. It is a server that uses REST API endpoints to do the same as above. However instead of saving to a `JSON` file, it uses `PostgreSQL` to save entries into a database.

### 02-jwt-golang

This project centred around understanding `JWT` in `GO`.
